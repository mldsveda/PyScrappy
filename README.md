## Functionality of the Scrapper

- Flipkart Scrapping
- Snapdeal Scrapping
- Wikipedia Scrapping
- Image Scrapping
- YouTube Scrapping
- Instagram Scrapping
- Alibaba Scrapping

## Usage

- Make sure you have Python installed in your system.
- Run Following command in the CMD.
 ```
  pip install PyScrappy
  ```
